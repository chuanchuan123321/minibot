# Web Skill

Techniques for effective web searching and information gathering.

## Search Strategies

### Effective Search Queries

- Use specific keywords related to your topic
- Include quotes for exact phrases: `"exact phrase"`
- Use minus sign to exclude terms: `-exclude_word`
- Use site: to search specific websites: `site:github.com`
- Use filetype: to find specific file types: `filetype:pdf`

### Search Operators

```
"exact phrase"          - Search for exact phrase
-word                   - Exclude word from results
site:domain.com         - Search only on specific site
filetype:pdf            - Search for specific file type
intitle:keyword         - Search in page title
inurl:keyword           - Search in URL
related:url             - Find related pages
cache:url               - View cached version
```

## Information Gathering

### Documentation

- Check official documentation first
- Look for README files in repositories
- Search for API documentation
- Check GitHub wikis and discussions

### Code Examples

- Search GitHub for example code
- Look for Stack Overflow answers
- Check official tutorials
- Review open source projects

### News & Updates

- Check official blogs
- Follow project announcements
- Monitor GitHub releases
- Subscribe to newsletters

## Best Practices

1. **Start broad, then narrow down** - Begin with general searches, then refine
2. **Check multiple sources** - Verify information from different sources
3. **Look for recent information** - Add year to search for current info
4. **Use advanced operators** - Combine operators for precise results
5. **Read carefully** - Skim results to find most relevant information

## Common Searches

- `"project name" documentation`
- `"error message" solution`
- `"technology" best practices 2024`
- `"library" github examples`
- `"problem" stack overflow`
